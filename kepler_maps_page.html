About This Dashboard

This dashboard demonstrates how to integrate multiple Kepler.gl maps into a single responsive webpage. The maps show pizzeria locations in Rome, extracted from OpenStreetMap data.

Features: Responsive design, modern card-based layout, smooth animations, and side-by-side comparison of different map views.

Data Source: OpenStreetMap via Overpass API using the query: amenity=restaurant AND cuisine~pizza
